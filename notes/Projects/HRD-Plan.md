# HRD Plan Survey

NIDA Well-being Survey platform built with vanilla HTML/JS + Supabase + Vercel static hosting.

## Objectives
- Survey well-being across organization
- Track employee health metrics
- Generate insights for HR planning

## Key Features
- [ ] Public well-being survey form
- [ ] Admin dashboard (CH1)
- [ ] Organization HR tools
- [ ] Real-time data analytics

## Tech Stack
- **Frontend**: Vanilla HTML/JS/CSS
- **Database**: Supabase PostgreSQL
- **Hosting**: Vercel (auto-deploy from main)
- **Architecture**: SSOT with form defs + org metadata
