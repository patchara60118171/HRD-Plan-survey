ระบบสำรวจ Well-being และ CH1 องค์กร
1
ภาพรวม
ยินดีต้อนรับสู่ Org HR Portal

หน้านี้คือ ศูนย์กลางการทำงานของ HR องค์กร สำหรับติดตามสถานะ CH1 ผลสำรวจ Well-being และแชร์ลิงก์ให้บุคลากรกรอกแบบประเมิน

  • ☰
    แถบเมนูซ้าย เลือกหน้าที่ต้องการใช้งาน — คลิกรายการเมนูเพื่อสลับ
  • 🔤
    แถบบน แสดงชื่อเมนูปัจจุบันและคำอธิบายสั้น ๆ
  • 📊
    หน้าหลัก (หน้าแรก) ดูภาพรวมความคืบหน้า CH1 และจำนวนผู้ตอบแบบสำรวจ Well-being
2
เมนูหลัก
เมนูที่ต้องใช้บ่อย
  • 🏠
    หน้าหลัก ดูสถานะการส่ง CH1 และจำนวนผู้ตอบ Well-being รวมทั้งหมด
  • 🏛
    รายละเอียดองค์กร ตรวจสอบและแก้ไขข้อมูลหน่วยงาน ข้อมูลผู้ประสานงาน
  • 📋
    CH1 ข้อมูลองค์กร เข้าถึงฟอร์ม CH1, ดูข้อมูลที่บันทึก, และดาวน์โหลด PDF
  • 📈
    Well-being ผลสำรวจ ดู Dashboard สรุปภาพรวมองค์กรจากแบบสอบถาม Well-being
  • 👤
    Well-being รายบุคคล ดูข้อมูล Raw Data รายคน และ Export เป็นไฟล์ Excel
  • 🔗
    แชร์ลิงก์ Well-being คัดลอกลิงก์แบบสำรวจและดาวน์โหลด QR Code สำหรับกระจายให้บุคลากร
3
แนวทางการใช้งาน
ขั้นตอนแนะนำ
1
เริ่มจาก "แชร์ลิงก์ Well-being" คัดลอกหรือดาวน์โหลด QR Code เพื่อส่งให้บุคลากรในหน่วยงานกรอกแบบสำรวจ
2
ติดตามผลที่ "Well-being ผลสำรวจ" ดู Dashboard สรุปตัวชี้วัดภาพรวมองค์กร เมื่อมีผู้ตอบเพียงพอ
3
ดูรายละเอียดที่ "Well-being รายบุคคล" เจาะลึกข้อมูลรายคนและ Export ข้อมูลออกมาวิเคราะห์เพิ่มเติม
4
อัปเดตแผนงานที่ "CH1 ข้อมูลองค์กร" กรอกหรือแก้ไขข้อมูลสถานะโครงการ HRD ขององค์กร และดาวน์โหลดเป็น PDF
💡
เคล็ดลับ: หากต้องการตรวจสอบหรืออัปเดตข้อมูลผู้ประสานงาน ให้ไปที่เมนู "รายละเอียดองค์กร" เพื่อแก้ไขข้อมูลก่อนส่งรายงาน
