# คู่มือสำหรับผู้ดูแลระบบ (Admin Guide)

## 1. การดูข้อมูลผู้ตอบแบบสอบถาม (Survey Data)

ข้อมูลทั้งหมดที่ผู้ใช้กด "ส่ง" จะถูกเก็บรวบรวมไว้ที่ **Google Sheets**

### วิธีเข้าดูข้อมูล
1.  เข้าไปที่ Google Drive ของบัญชีที่คุณใช้สร้าง Google Apps Script
2.  ค้นหาไฟล์ **Google Sheet** ที่คุณสร้างไว้ (เช่นชื่อ "Survey Responses")
3.  ข้อมูลจะเรียงบรรทัดลงมาตามลำดับเวลาที่ส่งเข้ามาล่าสุด

### ข้อมูลใน Google Sheet ประกอบด้วย
- **Timestamp**: เวลาที่ส่งข้อมูล (วัน/เดือน/ปี เวลา)
- **Email/Name**: ข้อมูลระบุตัวตน (ถ้า Login)
- **คำตอบรายข้อ**: แยกตาม Column (เช่น `sleep_bedtime`, `stress_level`)
- **ค่าที่คำนวณได้**: เช่น BMI, คะแนนซึมเศร้า (PHQ)

---

## 2. การนำข้อมูลไปใช้งานต่อ (Export)

คุณสามารถนำข้อมูลออกจาก Google Sheet ไปวิเคราะห์ต่อได้ง่ายๆ:

### วิธี Download เป็น Excel
1.  เปิดไฟล์ Google Sheet นั้น
2.  ไปที่เมนู **ไฟล์ (File)** > **ดาวน์โหลด (Download)**
3.  เลือก **Microsoft Excel (.xlsx)**

---

## 3. การแก้ปัญหาเบื้องต้น

### กรณีข้อมูลไม่เข้า Sheet
1.  ตรวจสอบว่า Deployment ID ในไฟล์ `js/app.js` ตรงกับที่ Deploy ล่าสุดหรือไม่
2.  ลองกด Deploy ใหม่ใน Apps Script Editor:
    - เลือก **Deploy** > **Manage deployments**
    - กด ✏️ (Edit)
    - ตรง Version เลือก **"New version"** แล้วกด Deploy
    - (URL เดิมจะยังใช้ได้ หากทำถูกวิธี)

### การแก้ไขคำถาม/ตัวเลือก
- หากแก้ไขคำถามใน schema กลาง (`data/questions-wellbeing.json`) ชื่อคอลัมน์ใน Google Sheet จะเพิ่มขึ้นมาใหม่เองอัตโนมัติเมื่อมีคนตอบเข้ามา
- ไม่จำเป็นต้องลบ Sheet เดิม ระบบจะเติม Column ใหม่ต่อท้ายให้เอง
