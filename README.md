# Well-being Survey / แบบสำรวจสุขภาวะบุคลากร

A web-based Individual Well-being Survey covering 4 dimensions of well-being for Thai organizations.

## Features
- 📋 4 dimensions: Physical, Mental, Social, Environment
- 🇹🇭 Thai language (simplified for general public)
- 💾 Auto-save to localStorage
- 📊 BMI & Depression score calculation
- 📥 Export to Excel
- 🔐 Google OAuth login

## How to Use
1. Open `index.html` in browser, or
2. Run local server: `python -m http.server 8080`
3. Go to http://localhost:8080

## Setup Google Login (Optional)
1. Create project in Google Cloud Console
2. Setup OAuth consent screen
3. Create OAuth Client ID
4. Add your client ID to `js/app.js`
