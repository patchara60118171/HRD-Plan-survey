Post-test Verification
ตรวจสอบข้อมูลใน Supabase
  • ส่งฟอร์มครบทั้ง 3 องค์กร TPSO · DSS · DHSS — ส่งสำเร็จและได้หน้ายืนยัน
    3 entries
  • org_id ถูกต้องตาม URL ตรวจคอลัมน์ org_id ว่าตรงกับ tpso / dss / dhss ตาม query param
    URL scope
  • BMI คำนวณถูกต้อง TPSO = 22.9 · DSS = 20.2 · DHSS = 23.4 (ไม่ใช่ null หรือ 0)
    bmi col
  • TMHI Score = 45 ตรวจคอลัมน์ tmhi_score ทั้ง 3 แถว ต้องเท่ากับ 45
    mental
  • UCLA Score = 60 ตรวจคอลัมน์ ucla_score ทั้ง 3 แถว ต้องเท่ากับ 60
    loneliness
  • ข้อมูล Array เก็บถูกต้อง คอลัมน์ที่เป็น array เช่น โรคประจำตัว, อาการ PM2.5, สิ่งรบกวน ต้องเป็น JSON array ไม่ใช่ string
    jsonb
  • ภาษาไทยแสดงถูกต้อง ตรวจว่าไม่มี mojibake หรือ character encoding ผิดพลาด
    UTF-8
  • ครบ 105 คอลัมน์ นับจำนวนคอลัมน์ใน survey_responses ว่าไม่ขาดคอลัมน์ใด
    105 cols
