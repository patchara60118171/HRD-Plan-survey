# Handoff Specialist Skill

## 🎯 Purpose
สร้างรายงานส่งมอบงานให้ Claude Cowork ดำเนินการต่อ พร้อมคำสั่งที่ชัดเจนสำหรับการก็อป-วาง

## 🔄 Handoff Process

### 1. งานเสร็จ → สร้าง Handoff Report
เมื่อ developer ทำงานส่วนใดสำเร็จ ให้สร้างรายงานส่งมอบพร้อมคำสั่งสำหรับ cowork

### 2. ส่งให้ Cowork
- ก็อป handoff report ทั้งหมด
- วางใน chat กับ cowork
- Cowork จะดำเนินการต่ออัตโนมัติ

```

## 🔄 งานที่ Cowork ต้องทำต่อ
```

## 🚀 วิธีการใช้งาน

### สำหรับ Developer:
1. **ทำงานส่วนตัวเสร็จ** (frontend, programming, bug fixes)
2. **เลือก Template** ที่เหมาะสม
3. **กรอกข้อมูล** ใน template
4. **ก็อปทั้งหมด** และวางใน chat กับ cowork
5. **Cowork จะทำงานต่อ** อัตโนมัติ

### สำหรับ Cowork:
เมื่อได้รับ handoff report:
1. **อ่านและเข้าใจ** สิ่งที่ต้องทำ
2. **ตรวจสอบ** ขอบเขตบทบาท (configuration only)
3. **ดำเนินการ** ตามคำสั่ง
4. **รายงาน** ผลลัพธ์กลับ
