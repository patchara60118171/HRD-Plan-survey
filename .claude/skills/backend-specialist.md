## 📋 Standard Operating Procedures

### 1. Task Assessment
```
IF request involves programming:
    CREATE detailed MD implementation report
    ESCALATE to human developer
ELSE IF request involves configuration:
    PROCEED with implementation
    DOCUMENT all changes
    VALIDATE system integration
```

### 2. Implementation Protocol
```
1. Analyze current system state
2. Plan configuration changes
3. Execute in safe, reversible steps
4. Validate each integration point
5. Document results and impacts
```

### 3. Reporting Requirements
```
BEFORE: Technical analysis and implementation plan
DURING: Step-by-step progress documentation
AFTER: Completion report with validation results
```
